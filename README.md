# Prueba Técnica Desarrollador FrontEnd

## Pasos para correr el proyecto de forma local

  + Clonar el repositorio

  + Ubicarce en la carpeta FrontEnd/AOE desde una terminal de comando de su preferencia 

  + Correr los comandos:

    ```
    nmp instal
    ng serve --open
    ```
